## 🚀 HOW TO GO LIVE ON GITHUB (FREE — 10 minutes)

### Step 1 — Create GitHub Account
1. Go to github.com
2. Click Sign up → create free account
3. Verify your email

### Step 2 — Create Repository
1. Click the + button top right → New repository
2. Name it: sky-blueprint
3. Set to: Public
4. Click: Create repository

### Step 3 — Upload Your Files
1. Click "uploading an existing file"
2. Upload all 3 files: index.html, style.css, app.js
3. Click "Commit changes"

### Step 4 — Enable GitHub Pages
1. Click Settings tab
2. Left menu → Pages
3. Source → Deploy from a branch
4. Branch → main → / (root) → Save
5. Wait 3 minutes

### Step 5 — Your site is LIVE!
Visit: https://YOUR-USERNAME.github.io/sky-blueprint

## 💳 HOW TO RECEIVE MONEY VIA PAYSTACK

### Step 1 — Create Paystack Account
1. Go to paystack.com
2. Click "Create free account"
3. Fill in your business details
4. Verify your email and phone

### Step 2 — Complete KYC (Identity Verification)
1. Dashboard → Settings → Business
2. Upload: SA ID document or passport
3. Upload: Proof of bank account
4. Wait 1-3 days for approval

### Step 3 — Add Your Bank Account
1. Paystack Dashboard → Settings → Bank Account
2. Add your South African bank account
3. Paystack pays you every day automatically

### Step 4 — Create Subscription Plans
1. Paystack Dashboard → Products → Plans
2. Create plan: "Sky Blueprint Pro" → R99/month
3. Create plan: "Sky Blueprint Business" → R249/month
4. Copy the Plan Codes they give you

### Step 5 — Update the App
In app.js, replace these lines with your real keys:
- PAYSTACK_PUBLIC_KEY = 'pk_live_YOUR_REAL_KEY'
- PLAN_CODES = { pro: 'YOUR_PRO_PLAN_CODE', business: 'YOUR_BUSINESS_PLAN_CODE' }

### Step 6 — Test First!
Use Paystack TEST keys first (pk_test_...) to test payments
Then switch to LIVE keys (pk_live_...) when ready
